### Compiling the Source Code

> Blockquote ./gbacc main.c activeBullet.s stagger.s

----------


### Default Controls

**Z** - Shoot / Start
**Up-Arrow** - Jump
**L-Arrow** - Move Left
**R-Arrow** - Move Right
**X** - Exit Game (Continue Screen)


----------


### Playing the Game
**Requirements**
Gameboy Advance Emulator (designed for VirtualBoy Advance)
*That's it!*

**Instructions**
 1. Open **bad-mario.gba** with your GBA Emulator
 2. Enjoy the game.
 3. Ragequit.
